# AJAX & APIs

- `books.js` — Google Books API: `fetchBook(queryType, queryValue)` prints title/author/ISBN for each result (Exercises 1-3). Run with `node books.js` or in the browser.
- `index.html` + `giphy.js` — GIF generator using the GIPHY API (Exercises 4-5).

## GIPHY key
Get a free key at https://developers.giphy.com/dashboard/ and paste it into `giphy.js` (`API_KEY`).
