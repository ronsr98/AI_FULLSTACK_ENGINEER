# Intro & JSX

JSX practice (`npm install` then `npm run dev`).

- **Exercise 1** - `showCompany(name, revenue)` returns a `div` per company; the `companies` array is mapped in the JSX.
- **Exercise 2** - `getClassName(temperature)` returns `freezing` / `fair` / `hell-scape`; the `#weatherBox` changes color by class (see `App.css`). Change the `temperature` variable to see different colors.
