# Module 3 - React

React assignments from the **Fullstack AI Engineer** course, built with **Vite + React 18** (function components & hooks).

## How to run any project
```bash
cd "<project folder>"
npm install
npm run dev
```
Then open the URL Vite prints (usually `http://localhost:5173`).

> `node_modules` is gitignored, so run `npm install` once per project after cloning.
